# static_mirrors

## Установка необходимых приложений

Предполагается, что данное приложение запускается в Ubuntu.
Для установки необходимых зависимостей выполните:
```bash
make init
```
Эта команда обновит репозитории системы, установит `unzip`, `wget`, `terraform`.
Выполнение займет несколько минут. 
Проверить корректность установки `terrafrom` можно командой
```bash
terraform --help
```
Если вывод будет содержать список возможных комманд - установка прошла успешно.

Кроме этого, необходимо установить `docker`.
## Получение всех языковых версий статьи

Для получения всех языковых версий статьи выполните
```bash
make sync
```
Скачанные языковые версии статьи будут помещены в `/sync`.

## Создание контейнера и размещение на зеркале

Для создания контейнера, его размещения и запуска в локальном docker выполните команду:
```bash
make deploy
```
После выполнения данной команды в удобном для вас браузере откройте адрес: `http://localhost:8000/bugs.html`
